# 零基础速成备考文档生成器

根据用户提供的任意学习资料，生成可直接跟着学、跟着刷题的备考文档。

---

## 一、适用场景

### 触发条件

用户表达以下意图时触发：
- "帮我生成备考文档/复习文档"
- "我零基础，需要速成"
- "距离考试X天，帮我总结"
- "基于这些资料帮我整理考点"

### 典型用户

| 用户类型 | 特点 | 核心需求 |
|---------|------|---------|
| 零基础学生 | 没学过或忘了 | 通俗易懂 |
| 临时抱佛脚 | 考前几天 | 抓重点高效 |
| 有资料没头绪 | 不知从何下手 | 系统整理 |

---

## 二、输入处理

### 收集信息

向用户确认：
1. 课程名称
2. 距离考试时间
3. 有哪些资料（复习要点/课件/题目/笔记等）
4. 文件路径

### 资料处理策略

| 资料类型 | 优先级 | 处理策略 |
|---------|:---:|---------|
| 复习要点/考纲 | ⭐⭐⭐ | 作为考点框架 |
| 历年真题/测验 | ⭐⭐⭐ | 统计频率，拆解题目 |
| 课件/课本 | ⭐⭐ | 提取核心概念 |
| 笔记 | ⭐ | 整理补充 |
| 无资料 | - | 生成通用框架 |

### 输入组合

| 输入组合 | 文档侧重点 |
|---------|-----------|
| 要点+测验 | 考点框架+题目拆解 |
| 只有要点 | 按要点展开解释 |
| 只有题目 | 从题反推考点 |
| 只有课本 | 提取核心知识 |
| 无资料 | 课程大纲+通用建议 |

---

## 三、输出规范

### 文件命名

[课程名称]零基础速成备考文档.md

### 文档结构

    # [课程]零基础速成备考文档

    # 📋 开篇
    - 题型分布（有题目时）/ 考点概览

    # 📋 结尾
    - 必背公式
    - 易错点速查
    - 考前速记

## 四、内容规范

### 考点定义

必须包含：
1. 专业术语的解释
2. 与相关概念的区别（易混时）

示例：

    **卡诺图**：用图形化简逻辑函数的方法。

    **与公式法区别**：
    | 方法 | 优点 | 缺点 |
    |------|------|------|
    | 卡诺图 | 直观 | 变量多麻烦 |
    | 公式法 | 灵活 | 需要技巧 |

### 解题步骤

必须做到：
1. 每步说明「做什么」和「为什么」
2. 不能跳步
3. 关键判断点解释清楚
4. 易错点特别标注

示例：

    1. **列出真值表**：2ⁿ种输入组合都写出
       - 3变量→8行，4变量→16行

    2. **写出表达式**：输出为1的项相加

    3. **化简**：卡诺图或公式法

### 通用解法模板

必须包含：
1. 明确步骤顺序
2. 每步判断条件
3. 常见变体处理
4. 结果验证方法

示例：

    **卡诺图化简通用解法**：

    步骤1：确定变量数，画对应大小卡诺图
      - 2变量：2×2，3变量：2×4，4变量：4×4

    步骤2：填入最小项（输出为1填1，无关项填×）

    步骤3：圈选
      - 每个圈2ⁿ个格子（1/2/4/8...）
      - 必须矩形，每个1至少被圈一次
      - 圈越大越好，圈数越少越好

    步骤4：写结果
      - 不变的变量保留，变化的消去

### 对比表格

易混概念必须用表格：

    | 特性 | 同步置数 | 异步清零 |
    |:---:|:---|:---|
    | 触发 | 时钟+置数信号 | 仅清零信号 |
    | 需要时钟 | ✅ | ❌ |
    | 速度 | 较慢 | 快 |

## 八、注意事项

### 内容边界

| ✅ 应该 | ❌ 不应该 |
|---------|-----------|
| 基于用户资料 | 引入外部知识 |
| 解释已有概念 | 扩展超纲 |
| 标注高频考点 | 预测原题 |
| 提供解题模板 | 替代学习 |
| 灵活调整结构 | 生搬硬套 |

### 零基础原则

1. **术语解释**：首次出现必须解释
2. **步骤完整**：不跳步，说明做什么和为什么
3. **对比区分**：易混概念用表格对比
4. **记忆辅助**：提供口诀和技巧

### 特殊情况

| 情况 | 处理方式 |
|------|---------|
| 资料有错误 | 指出并给正确答案 |
| 资料不完整 | 补充说明，标注来源 |
| 要求超纲 | 提醒并询问是否继续 |
| 图片无法处理 | 说明限制，请用户描述 |
